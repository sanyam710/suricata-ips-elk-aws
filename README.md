Cloud-Based Intrusion Prevention System using Suricata and ELK Stack
Overview

This project demonstrates the implementation of a Network Intrusion Prevention System (IPS) designed to protect a web server hosted on Amazon Web Services (AWS).

The solution uses Suricata in inline IPS mode to inspect network traffic and block malicious requests before they reach the backend web server. Security events are collected and analyzed using the Elasticsearch–Kibana stack, enabling real-time monitoring and visualization of attack patterns.
